乔木阅读记录（4月5日）

  • 想赚1000万，也要从给用户带来1块钱价值的事情做起。

  • 用户的第一印象非常重要：首页、着陆页、注册、登录、客服等所有跟用户首次接触的地方都要注意。

  • 喜欢主动买单的人不是因为钱多，而是因为把友情看的比金钱重要。

  • 工作时愿意多做的人不是因为傻，而是懂得承担责任。

  • 吵架后先道歉的人不是因为错，而是懂得珍惜身边的人。

  • 愿意帮助你的人不是因为欠你什么，而是把你当真朋友。

  • 要像谈恋爱一样去跟用户打交道：注意第一印象，获得第一次的好感；

  • 需要Review上周团队人员的工作完成情况，没完成的是否遇到什么阻力需要帮助。

  • 在客户评价里面增加情绪选项，就像给客户面前摆了一面镜子，让他能看清自己的表情，自然会镇定自己的情绪。

  • 转化和流失之间就是增长，减少客户流失比增长新用户来的要容易的多。

  • 雇你喜欢的人，可以让你每天都想一起工作的人。

  • 超过15分钟的事情都要拿到会议上去专门讨论。

  • 获取新用户就像约会追女孩，留住老用户就像成功的婚姻。

  • 精心制作每个帮助页面，让用户出现问题时候能轻易找到精确的帮助。

  • 最实用的占领市场的方法：1.最好的价格 2.最好的产品 3.最好的整体解决方案

  • 每个人都参与当客服，敦促大家一起改进问题，优化产品和内部工具。

  • 满意的用户愿意给更多的建议和称赞（包括钱），舒心的团队更倾向于服务好用户和做出更好的产品。

  • 对待产品不能仅仅满足于做到“基本可用”，而要做到“有魅力”，让用户发自内心喜爱产品。

  • “CEO轮值法”可以让团队成员从宏观上理解公司的核心业务，一天内调动所有公司资源做出他认为激动人心的功能。

  • 一个产品从不了解到熟练掌握之间有很长的距离，但只要让用户更快上手达到基本熟练就可以。
