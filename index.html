Hexo博客后台运行技巧

安装并运行hexo博客后，一般教程都会告诉运行命令是:hexo s,但是关掉Terminal，进程也就结束了，对于一个希望永远运行的服务器来说是不可接受的。

查阅一些资料后找到一个后台运行的命令: hexo s &

这个命令确实可以实现后台运行，但是从我的使用情况来看，每天hexo的服务进程都会死掉，还找不到原因。

于是不得不使用nodejs里很流行的forever，可以实现当进程死掉时候自动重启。Forever全局安装命令
npm install -g forever

安装完成，还不能直接用，需要在你的hexo根目录下新建一个js文件，比如app.js,代码如下：

require(‘hexo’).init({command: ‘server’});

之后使用命令forever start app.js 即可。

查看forever进程命令 forever list, 更多命令输入forever -h

让网站飞一般快还省钱，快用这些CDN库！

网站刚换了新机房，速度飞快，但是发现网站的CSS、JS文件耗费了大量带宽。

对于现代的网站开发，几乎都会使用很多第三方开源库，比如bootstrap、jquery、font-awesome等，这些静态文件如果存自己服务器，一方面浪费带宽多花钱，另一方面需要CDN加速。

除此之外，浏览器还有同一个域名下http请求数并发连接数限制，超过后会阻塞，让网页打开变慢，还有第三方公共库使用的网站多，等用户访问你的网站时候可能已经被浏览器缓存过了，让你的网站打开更快！

所以第三方静态文件CDN加速很有意义和价值，那怎么找好的CDN库呢？

不少大的互联网公司为了维护帮助自己的开发者，都制作了CDN公共库，比如百度、360、新浪等，还有一些做云存储的公司，为了提升品牌知名度、好誉度，也都做了这件事情，比如七牛云存储、又拍云等。

推荐列表如下：

1.百度静态资源公共库 http://cdn.code.baidu.com/

品牌够大，心理上放心些，库内容相对较多，查找也比较方便。

2.Bootcss网站与又拍云合作的CDN http://www.bootcdn.cn/

库内容非常丰富，就连很多不常用的Jquery插件都可以找到，而且方便拷贝链接，用户体验好。

3.七牛云存储提供的CDN http://www.staticfile.org/

库内容非常丰富，还贴心的提供了npm工具方便查找下载、Copy地址。

4.360网站卫士常用前端公共库CDN服务 http://libs.useso.com/

常用的文件都有，还提供谷歌字体。

5.新浪提供的库 http://lib.sinaapp.com/

比较老旧，但据说很稳定

6.百度开放平台CDN公共库 http://developer.baidu.com/wiki/index.php?title=docs/cplat/libs

又一个百度的库，可能给开放平台用户使用的，内容比较少，只有常用的，版本也不全。

7.CDNJS网站提供的库，与cloudflare云平台合作。 https://cdnjs.com/

理论上是最好的库，但因为是国外的，不定哪天被天朝墙，国内很多库都镜像这里的。

以上CDN排序只是个人喜好，大家自己尝试，选择最适合自己的，建议从品牌、稳定性方面综合考虑。

快用起来，让你的网站快的飞起！
