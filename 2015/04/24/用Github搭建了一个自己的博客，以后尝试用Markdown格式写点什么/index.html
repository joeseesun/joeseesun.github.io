用Github搭建自己的带域名博客步骤要点

  • github
  • 技术

  1. 先注册个GitHub账号，http://www.github.com

  2. 在github上新建一个库，注意格式：username.github.io,username就是你的github用户名，只有建立这样的库才可以。

  3. 本地电脑安装Nodejs和Git，安装hexo命令 npm install hexo -g

  4. 在电脑上建个文件夹，然后运行命令hexo init，然后运行npm install

  5. 输入hexo s,本地的博客就搭建好了，访问地址 http://localhost:4000 即可。

  6. 写文章命令 hexo n "标题名"，会自动在Source文件夹下的_post文件夹下建一个md格式的文档，用sublime打开编辑。

  7. 修改根目录下的_config.yml文件，配置git发布地址，博客名称等信息。

    deploy:
    type: github
    repository: https://github.com/username/usernmae.github.io.git
    branch: master

  1. 运行命令 hexo d就发布到github了。

  2. 为了支持域名绑定，需要在source目录下建一个名字叫 CNAME的文件，里面写上你要绑定的域名，比如我的 xiexiefa.com,域名那边只需要建一个CNAME记录，指向 username.github.io(不要加http://)

  3. 这样博客就搭建好了。
