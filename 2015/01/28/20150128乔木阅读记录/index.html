乔木阅读记录（1月28日）

  • 阅读记录

  • 运营的核心:吸引用户、留住用户、发展用户。

  • 一个不够强的需求，不能让人坚持使用的东西如智能手环，但可以在购买一个本来就要买的产品时，选择一个附带有这个能力的智能产品。

  • 在未形成核心差异价值之前，做附缀，而不是新增。

  • 人类天然是群体生存的动物，害怕被群体遗忘，害怕被孤立，所以在带着手机后，便总是害怕自己丢失了最新通知，回复慢了，错过电话/微信。

  • 通知焦虑是现代人几乎人人都有，但很难意识到的一个精神问题。

  • 奥格威曾经说，不要设计不愿给自己家人看到的广告。

  • Code Review机制实行的三个要素：编码规范（可参考Google的）、代码评审平台（如Gerrit）、不通过审核不能入库的强制规定。

  • 定位作为现代营销学中最有影响力的理论，得益于它充分的利用了人类大脑生理上的特点或者是缺陷。

  • 但凡需要差异化来生存的企业，比市场更残酷的地方，是顾客们懒惰的心。时刻谨记定位为本，攻心为上，才是唯一的出路。
